# 数字图像处理基础

本目录用于学习数字图像的像素表示、ISP 处理流程，以及 Gamma 校正。

## 目录内容

- `1数字图像获取.md`：数字图像、RAW/RGB/YUV 和 ISP 流程笔记。
- `code/`：Gamma 校正示例及测试图片。
- `*.png`：笔记中使用的示意图。

## 使用方法

1. 先阅读 `1数字图像获取.md`，了解数字图像和 ISP 的基础概念。
2. 进入 `code/`，按照其中的 `README.md` 运行 Gamma 校正示例。

运行代码前需要安装：

```powershell
python -m pip install numpy opencv-python
```
