# 一.什么是图像直方图
图像像素强度分布的图形表示，对灰度图像，直方图显示每个灰度级出现的频率；对彩色图像，可以分别计算每个通道的直方图
# 二.直方图有什么用
- 亮度与对比度
  - 如果峰值集中在左边，说明图像偏暗，反之亦然
  - 如果峰值分布很窄，说明图像对比度较低，反之亦然
- 内容分析：可以大致推断图像内容，比如大部分是天空，那B通道在某区域可能有高峰
- 图像阈值化：可以选择合适的阈值，用于将图像分割成前景和背景
- 图像均衡化：通过拉伸直方图，使其分布更均匀，增强图像对比度
- 图像匹配与检索：比较不同图像的直方图，可以简单度量图像相似性
openCV相关直方图操作

| 功能         | 函数                       | 说明                         |
| ------------ | -------------------------- | ---------------------------- |
| 计算直方图   | `cv::calcHist()`           | 计算图像的直方图。           |
| 直方图均衡化 | `cv::equalizeHist()`       | 增强图像的对比度。           |
| 直方图比较   | `cv::compareHist()`        | 比较两个直方图的相似度。     |
| 绘制直方图   | `matplotlib.pyplot.plot()` | 使用 Matplotlib 绘制直方图。 |

遍历图像的每个像素，将像素值对应的数组位置加1，然后在空白图像上绘制柱状图。
#### task2
- 统计每个灰度频数
- 计算累积分布函数cdf，即每个灰度级和以下所有灰度级的累积概率
- 映射公式 $S_k$ =round(255×CDF( $r_k$ ))
- 将映射应用到原图的每个像素
#### task3
- 计算源图像和参考图像的各自累积分布函数( $CDF_s$ 和 $CDF_r$ )
- 构建映射表：对每个灰度级i，在 $CDF_r$ 中找到一个灰度级j，使得| $CDF_s[i]$ - $CDF_r[j]$ |最小
- 将映射应用到源图像
让源图模仿参考图的亮度、对比度和色彩风格。把源图中处于某排名的累计灰度，替换为参考图中相似排名位置的灰度。
